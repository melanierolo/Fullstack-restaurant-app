# Fullstack-restaurant-app

## Description

This project was successfully carried out and it was develop for the course Full Stack Development with MERN (MITxPro)

## Technology used

### Frontend

<ul>
  <li>Nextjsv12</li>
  <li>Next-auth</li>
  <li>Reactstrap</li>
  <li>Graphqlv16</li>
</ul>

### Backend

<ul>
  <li>Strapiv4</li>
  <li>Stripev9</li>
  <li>Better-sqlite3</li>
  <li>Provider-upload-cloudinary</li>
</ul>
